# 日本語
## ザリマップ
こちらはザリマップのサイトデータを取り扱っております。<br>
無断複製はおやめ下さい。

## ザリマップの使用方法
直感的に操作できるように設計しています。<br>
マップのピンまたは詳細パネルの情報をクリックすると、詳細が表示されます。

# English
## Zarimap
This site hosts the data for Zarimap.<br>
Please do not reproduce the content without permission.

## How to Use Zarimap
It is designed for intuitive use.<br>
Click on a map pin or the information in the details panel to view more details.
